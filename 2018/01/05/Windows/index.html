Windows常用命令以及快捷键

发表于 2018-01-05

常用命令

系统类

  1. regedit 打开注册表编辑器
  2. cmd 打开DOS运行窗口
  3. msconfig 计算机启动管理
  4. control 打开控制面板
  5. slmgr.vbs -xpr 查看系统激活信息
  6. explorer 打开资源管理器
  7. taskmgr 打开任务管理器
  8. dxdiag 查看电脑显卡 DirectX 系统信息等
  9. gpedit.msc 组策略命令,打开本地组策略编辑器
  10. appwiz.cpl 卸载或更改程序
  11. Compmgmtlauncher 计算机管理
  12. desk.cpl 屏幕分辨率
  13. devicepairingwizard 给当前计算机添加设备
  14. devmgmt.msc 设备管理器
  15. displayswitch 显示模式切换
  16. eventvwr 事件查看器
  17. fsmgmt.msc 打开共享文件夹
  18. lusrmgr.msc 本地用户和组
  19. netplwiz 用户账户管理
  20. slui Windows系统激活
  21. sndvol 系统音量合成器
  22. wscui.cpl 安全性与维护
  23. winhlp32 Windows 帮助和支持
  24. wf.msc 高级安全 Windows 防火墙
  25. sysdm.cpl 系统属性
  26. services.msc 系统服务管理
  27. sdclt 备份和还原
  28. rstrui 系统还原
  29. resmon 资源监视器
  30. powercfg.cpl 电源选项
  31. optionalfeatures Windows 功能管理
  32. ncpa.cpl 网络连接 网络连接管理
  33. computerdefaults 设置程序访问此计算机的默认值

工具类

  1. charmap 字符映射表
  2. cleanmgr 磁盘清理
  3. SnippingTool 打开截图工具
  4. calc.exe打开计算器
  5. mspaint 画图
  6. notepad 打开记事本
  7. write 打开写字板
  8. mstsc 远程桌面连接
  9. credwiz 备份系统用户名和密码
  10. nslookup IP地址侦测器
  11. osk 屏幕键盘
  12. perfmon 计算机性能监视器
  13. mdsched 内存诊断工具
  14. rekeywiz 文件加密系统
  15. stikynot便签工具
  16. shrpubw 共享文件夹创建向导
  17. mrt Microsoft Windows 恶意软件删除工具
  18. psr 步骤记录器（PPT可分步查看，也可以注释，亦可以在windows上播放）
  19. mip 数学输入面板
  20. magnify 放大镜
  21. dccw 显示颜色校准

常用快捷键

  1. CTRL+C 复制
  2. CTRL+V 粘贴
  3. CTRL+X 剪切
  4. CTRL+D OR DELETE 删除
  5. CTRL+A 全选
  6. CTRL+S 保存
  7. CTRL+F 查找
  8. CTRL+W 关闭当前窗口
  9. CTRL+Z 撤销
  10. CTRL+ ESC 显示开始菜单
  11. ALT+SHIFT+BACKSPACE 或CTRL+Y 重做上一步被撤销的操作
  12. CTRL+F4 关闭当前
  13. CTRL+TAB 切换当前选项卡向前移动
  14. CTRL+SHIFT+TAB 切换当前选项卡向后移动
  15. CTRL+SHIFT OR SHIFT OR WIN+空格 切换输入法
  16. CTRL+SHIFT+ESC 或 CTRL+ALT+DELETE打 开任务管理器
  17. F10+ALT 激活当前程序的菜单栏
  18. F1显示Windows帮助内容
  19. F2 重命名
  20. F5刷新
  21. ALT+SPACEBAR 打开程序最左上角的菜单
  22. ALT+TAB 切换当前程序 （加SHIFT反向）
  23. ALT+ESC 切换当前程序 （加SHIFT反向）
  24. ALT+ENTER 将windows下运行的MSDOS窗口在窗口和全屏幕状态间切换
  25. PRINT SCREEN 将当前屏幕以图像方式拷贝到剪贴板(笔记本上是PrtScSysRq)
  26. Alt+加下划线的字母 执行与该字母匹配的命令（或选择选项）
  27. Windows键+L 锁定计算机或切换用户
  28. Windows键+M 最小化所有窗口
  29. Windows键+R 打开“运行”对话框
  30. Windows键+PAUSEBREAK打开“系统属性”对话框
  31. Windows键+ 向上键最大化窗口（windowsXP不适用）
  32. Windows键+ 向左键将窗口最大化到屏幕的左侧（windowsXP不适用）
  33. Windows键+ 向左键将窗口最大化到屏幕的左侧（windowsXP不适用）
  34. Windows键+ 向右键将窗口最大化到屏幕的右侧（windowsXP不适用）
  35. Windows键+ Home最小化除活动窗口之外的所有窗口（windowsXP不适用）
  36. Windows键+ Shift + 向上键将窗口拉伸到屏幕的顶部和底部。（windowsXP不适用）
  37. Windows键+ Shift + 向左键或向右键将窗口从一个监视器移动到另一个监视器
  38. Windows键 + T 循环切换任务栏上的程序
  39. NUM LOCK+负号(-)折叠所选的文件夹
  40. NUM LOCK+乘号(*)展开所选的文件夹下的所有文件及文件夹
  41. NUM LOCK+加号(+) 展开所选的文件夹
  42. SHIFT+上下左右 选择光标上下左右的文字

CMD常用命令

  1. help

    查看CMD的原生命令

  2. 命令+/？ 或 命令 -help

    查看当前命令的用法

  3. F7按键 或 上下键

    查看使用过的命令

  4. Ctrl+C

    中断操作键

  5. >和>>

    文件重定向

  6. echo

    写入文件内容

  7. lable

    建立磁盘卷标

  8. vol

    显示磁盘卷标命令

  9. sandisk

    检测、修复磁盘命令

  10. defrag

    整理磁盘、消除磁盘碎块

  11. xcopy

    目录复制命令

  12. fc

    文件比较命令

  13. attrib

    修改文件属性命令

  14. shutdown

    关机命令

  15. nbstat

    使用TCP/IP的NetBIOS协议显示协议统计和当前的TCP/IP连接

  16. at

    在指定的时间运行命令

  17. ​

  18. ipconfig

    查看和管理当前计算机的网络配置

  19. route

    查看和管理当前计算机的路由表

  20. ping

    最常用的网络连通性测试命令

  21. systeminfo

    查看系统信息

  22. tasklist

    查看进程列表

  23. netstat

    查看网络状态

  24. net

    十分强大的命令，直接/?去学习吧

  25. tracert

    查看路由，更建议使用BestTracer或者WinMTR等工具，在linux下推荐MTR工具

  26. query

    查看进程

  27. dir

    查看文件夹列表

  28. md

    md 文件夹名称

    创建文件夹

  29. cd

    cd 路径

    进入路径 cd ..是返回上一级路径

  30. rd

    rd 目录名

    删除目录

  31. copy

    copy 路径/文件名1 路径/文件名2

    复制文件

  32. del

    del 文件路径

    删除文件

  33. ren

    ren 文件路径 文件名更改文件名称

  34. type

    显示文本文件

  35. format

    磁盘格式化

  36. discopy

    复制磁盘

  37. deltree

    删除目录树

  38. mem

    查看内存及使用情况

  39. chkdsk

    chkdsk 磁盘名

    检查磁盘使用情况

  40. sys

    系统文件传递命令

  41. path

    设定DOS寻找.COM.EXE.BAT文件的所在目录

  42. edit

    一个简单的编辑软件，可以用来编辑程序和一些批处理文件

  43. cls

    清除屏幕上的内容

  44. date

    显示当前系统时间

  45. ver

    显示系统版本号

  46. U盘病毒清除

    • 命令提示符窗口中切换到U盘所在盘符]或是中了Autorun.in病毒的盘符下，依次执行命令：attrib -S -H -R autorun.inf按回车
    • 再次执行命令：del autorun.inf
